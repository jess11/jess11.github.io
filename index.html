    Get Fashion

    E-commerce fashion website. This was a group project and I was responsible for creating the back-end for the pages showing the listing of the products and creating a filtering system, and connecting the user from the homepage, to the product listing and then to the individual products. I also implemented the Stripe api to allow credit card payments and contributed to the front end for the admin and payment pages.

    Created with HTML, CSS, Javascript, Bootstrap, Ruby on Rails, Stripe api, API.ai, Cloudinary

    Twitter Simplified

    A web app where people can sign up, share posts and follow each other. I tried to make a clone of Twitter.

    Image Maker

    Users can take photos of themselves, draw on it and then publish it onto the website. Also includes application of filters using a face detection library. Currently, doesn't work on iOS phones due to lack of support of webRTC.

    Created with HTML, CSS, Javascript, jQuery, webRTC, Bootstrap, Ruby on Rails, Cloudinary, CCV.js/face.js, ajax
